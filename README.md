# Coursework Software Development Theory

## About
Go project for creating and playing cartoon from a binary file.

![](preview.gif)

## Problem
- With chance `91%` create one scene 
- With chance `81%` create two scenes 
- Add music for scenes

## Project structure
```bash
src/
├── first_scene
│   ├── first_scene.go
│   ├── music.go
│   └── snowflake.go
├── second_scene
│   ├── firework.go
│   ├── music.go
│   └── second_scene.go
└── settings
│   └──  settings.go
├── go.mod
├── go.sum
├── main.go
```

## How run program
If you have Windows OS just run `bin\cartoon-win-x86.exe` and have a nice view!
